# Project Walk

**Team Name:** Palantir

A React Native/Expo walking app that rewards users with points for walking. Earn points based on steps taken (1 point per 100 steps) and redeem them for rewards.

## Tech Stack & Dependencies

- **Framework:** Expo SDK 54, React Native 0.81
- **Language:** TypeScript (strict mode)
- **UI:** React 19 with React Compiler
- **Routing:** expo-router (file-based routing)
- **Key Libraries:**
  - react-native-safe-area-context
  - expo-haptics
  - expo-symbols

## Setup Instructions

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the app:

   ```bash
   npm start
   ```

3. Run on specific platform:

   ```bash
   npm run android    # Android
   npm run ios        # iOS
   npm run web        # Web
   ```
